# 🎬 DramaNest

تطبيق دراما متقدم للهاتف المحمول - عالم الدراما بلا حدود

## ✨ الميزات

- 📺 **مكتبة ضخمة**: أكثر من 5000 مسلسل عربي وعالمي
- 💎 **خطط مرنة**: اختر الخطة المناسبة لك
- 💳 **دفع آمن**: طرق دفع متعددة (قريباً)
- 🌍 **عملات متعددة**: SAR و USD
- 🎨 **تصميم احترافي**: واجهة مستخدم جميلة وسهلة الاستخدام
- 🔐 **دعم RTL**: واجهة كاملة للعربية
- ❤️ **المفضلة**: احفظ مسلسلاتك المفضلة (قريباً)
- 📊 **ملف شخصي**: تتبع اشتراكاتك ومشاهداتك

## 🛠️ التكنولوجيا

- **React Native** - تطوير تطبيقات iOS و Android
- **TypeScript** - للتطوير الآمن
- **React Navigation** - إدارة الملاحة
- **AsyncStorage** - تخزين البيانات محلياً

## 📱 الشاشات

1. **الرئيسية** 🏠 - عرض المسلسلات الأكثر مشاهدة
2. **المسلسلات** 📺 - استعراض جميع المسلسلات
3. **الاشتراكات** 💎 - عرض الخطط المتاحة
4. **ملفي** 👤 - إدارة الحساب والإعدادات

## 🚀 البدء

### المتطلبات
- Node.js (v14 أو أعلى)
- npm أو yarn
- React Native CLI
- Android Studio (لـ Android) أو Xcode (لـ iOS)

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/eydinrustom-afk/DramaNest.git
cd DramaNest

# تثبيت الحزم
npm install
```

### التشغيل

#### على Android
```bash
npm run android
```

#### على iOS
```bash
npm run ios
```
